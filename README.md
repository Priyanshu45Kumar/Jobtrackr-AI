# JobTrackr AI 🚀

A full-stack MERN job application tracker featuring authentication, dashboard analytics, Kanban board with drag-and-drop workflow, and AI-powered career tools.

## Features

* 🔐 Authentication & Authorization
* 📊 Dashboard Analytics
* 📝 Applications CRUD
* 🔍 Search & Filter
* 📌 Kanban Board
* 🖱️ Drag & Drop Workflow
* 📅 Follow-up Tracking

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Axios
* React Router
* dnd-kit

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Mongoose

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` inside `backend`:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
